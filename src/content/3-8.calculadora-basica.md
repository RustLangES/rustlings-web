# Ejercicio: Calculadora Básica

Implementa las cuatro operaciones aritméticas básicas.

## Funciones a implementar

```rust
fn sumar(a: i32, b: i32) -> i32
fn restar(a: i32, b: i32) -> i32
fn multiplicar(a: i32, b: i32) -> i32
fn dividir(a: i32, b: i32) -> f64
```

## Ejemplos

```
sumar(2, 3)       → 5
restar(5, 3)      → 2
multiplicar(3, 4) → 12
dividir(10, 2)    → 5.0
```

> Implementa cada función y ejecuta los tests para verificar tu solución.
