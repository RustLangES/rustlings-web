# HashMap<K, V>

A `HashMap` stores key-value pairs. It's Rust's hash table.

## Create and populate

```rust
use std::collections::HashMap;

fn main() {
    let mut map: HashMap<String, i32> = HashMap::new();

    map.insert(String::from("one"),   1);
    map.insert(String::from("two"),   2);
    map.insert(String::from("three"), 3);

    // From two vectors with zip
    let keys   = vec!["a", "b", "c"];
    let values = vec![1, 2, 3];
    let map2: HashMap<_, _> = keys.iter().zip(values.iter()).collect();
}
```

## Accessing values

```rust
fn main() {
    let mut m = HashMap::new();
    m.insert("key", 42);

    // get — returns Option<&V>
    match m.get("key") {
        Some(v) => println!("Value: {v}"),
        None    => println!("Not found"),
    }

    // Direct access (panic if not present)
    println!("{}", m["key"]); // 42
}
```

## Common updates

```rust
fn main() {
    let mut m: HashMap<String, u32> = HashMap::new();

    // Overwrite
    m.insert(String::from("x"), 10);
    m.insert(String::from("x"), 20); // overwrites

    // Insert only if not present
    m.entry(String::from("y")).or_insert(5);
    m.entry(String::from("y")).or_insert(99); // y already exists, no change

    // Update based on previous value
    let counter = m.entry(String::from("z")).or_insert(0);
    *counter += 1;

    println!("{:?}", m);
}
```

## Practical case: word counting

```rust
use std::collections::HashMap;

fn count_words(text: &str) -> HashMap<&str, usize> {
    let mut counts = HashMap::new();

    for word in text.split_whitespace() {
        let c = counts.entry(word).or_insert(0);
        *c += 1;
    }

    counts
}

fn main() {
    let text = "hello world hello rust world hello";
    let counts = count_words(text);

    let mut pairs: Vec<_> = counts.iter().collect();
    pairs.sort_by(|a, b| b.1.cmp(a.1)); // sort by frequency

    for (word, n) in pairs {
        println!("{word}: {n}");
    }
}
```

## Ownership in HashMaps

Types that don't implement `Copy` are moved when inserted:

```rust
fn main() {
    let mut m = HashMap::new();

    let key   = String::from("key");
    let value = String::from("value");

    m.insert(key, value);

    // key and value no longer exist — they were moved
    // println!("{key}"); // ERROR

    // Solution: clone if you need to keep the variables
    let k = String::from("k");
    m.insert(k.clone(), String::from("v"));
    println!("{k}"); // OK
}
```

## Removing entries

```rust
fn main() {
    let mut m = HashMap::new();
    m.insert("a", 1);
    m.insert("b", 2);

    m.remove("a");
    println!("{:?}", m); // {"b": 2}

    m.retain(|_, v| *v > 1); // keeps only entries that satisfy the condition
    println!("{:?}", m);
}
```
