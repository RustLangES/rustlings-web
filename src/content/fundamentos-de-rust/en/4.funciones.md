# Functions

Functions are the fundamental building block of any Rust program. They are defined with `fn` and by convention use `snake_case`.

## Basic function

```rust
fn main() {
    println!("Hello, world!");
    another_function();
}

fn another_function() {
    println!("This is another function.");
}
```

In Rust, declaration order doesn't matter — you can call `another_function` before it appears in the code.

## Parameters and types

All parameters **must** have an explicit type:

```rust
fn greet(name: &str, times: u32) {
    for _ in 0..times {
        println!("Hello, {name}!");
    }
}

fn main() {
    greet("Alice", 3);
}
```

## Return value

The return type is indicated with `->`. The last expression of the function is the return value (without semicolon):

```rust
fn square(x: i32) -> i32 {
    x * x  // expression — no semicolon
}
```

Using explicit `return` is valid for early returns:

```rust
fn divide(a: f64, b: f64) -> f64 {
    if b == 0.0 {
        return 0.0;  // early return
    }
    a / b  // implicit return
}
```

## Statements vs Expressions

This distinction is key in Rust:

- **Statement:** performs an action, **does not return a value**. Ends with `;`.
- **Expression:** evaluates to a value. **Without semicolon**.

```rust
fn main() {
    let x = 5;              // statement

    let y = {
        let a = 3;
        a + 1               // expression — returns 4
    };                      // the block assigns 4 to y

    println!("y = {y}");    // y is 4
}
```

## Functions that return `()`

When a function returns nothing, it implicitly returns `()` (the "unit type"):

```rust
fn print(message: &str) {  // equivalent to -> ()
    println!("{message}");
}
```

## Returning multiple values with tuples

Rust doesn't have multiple returns directly, but tuples are idiomatic for this:

```rust
fn statistics(data: &[f64]) -> (f64, f64, f64) {
    let sum: f64 = data.iter().sum();
    let min = data.iter().cloned().fold(f64::INFINITY, f64::min);
    let max = data.iter().cloned().fold(f64::NEG_INFINITY, f64::max);
    (sum, min, max)
}

fn main() {
    let data = [3.0, 1.5, 4.0, 1.0, 5.5];
    let (sum, min, max) = statistics(&data);
    println!("sum={sum}, min={min}, max={max}");
}
```
