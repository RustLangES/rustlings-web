# if as an Expression

## Basic if

```rust
fn main() {
    let number = 6;

    if number % 4 == 0 {
        println!("Divisible by 4");
    } else if number % 3 == 0 {
        println!("Divisible by 3");
    } else if number % 2 == 0 {
        println!("Divisible by 2");
    } else {
        println!("Not divisible by 4, 3, or 2");
    }
}
```

Unlike many languages, the condition in Rust **does not need parentheses** and must be strictly `bool` — there's no implicit conversion from integers:

```rust
// In C: if (x) { ... }  ← valid
// In Rust:
// if x { ... }          ← ERROR if x is not bool
// if x != 0 { ... }     ← correct
```

## if as an expression

In Rust, `if` is an expression — it can return a value:

```rust
fn main() {
    let condition = true;
    let number = if condition { 5 } else { 6 };
    println!("The number is: {number}");
}
```

**Important rule:** both branches must return the same type:

```rust
// ERROR: incompatible types
// let value = if condition { 5 } else { "six" };

// Correct:
let value = if condition { 5 } else { 6 };
```

## Single-line vs multi-line if expressions

```rust
fn main() {
    let temperature = 22;

    // Concise for simple conditions
    let status = if temperature > 20 { "warm" } else { "cold" };

    // Multi-line for more complex logic
    let activity = if temperature > 30 {
        "go to the beach"
    } else if temperature > 20 {
        "go for a walk"
    } else {
        "stay home"
    };

    println!("Weather {status}: {activity}");
}
```

> **Tip:** Use `if` as an expression to assign values based on conditions. It's more idiomatic than using `let mut` with reassignments.
