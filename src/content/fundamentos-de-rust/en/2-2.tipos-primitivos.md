# Primitive Types

Rust is a **statically typed** language — all types are known at compile time. The compiler can infer them in most cases, but it's important to understand them.

## Integers

Rust offers signed and unsigned integers of various sizes:

| Length | Signed | Unsigned |
|--------|--------|----------|
| 8-bit  | `i8`   | `u8`     |
| 16-bit | `i16`  | `u16`    |
| 32-bit | `i32`  | `u32`    |
| 64-bit | `i64`  | `u64`    |
| arch   | `isize`| `usize`  |

```rust
fn main() {
    let a: i32 = -100;      // default for integers
    let b: u8  = 255;       // 0 to 255
    let c: u64 = 1_000_000; // underscores for readability
    let index: usize = 0;   // always for indexing collections
}
```

**Practical rule:** use `i32` for general-purpose integers and `usize` for indices.

## Floating point

```rust
fn main() {
    let x = 2.0;       // f64 by default
    let y: f32 = 3.0;  // f32 if you need to save memory

    let result = x * 3.14159;
    println!("{:.2}", result); // two decimal places
}
```

## Booleans

```rust
fn main() {
    let t = true;
    let f: bool = false;

    let and = t && f;  // false
    let or  = t || f;  // true
    let not = !t;      // false

    println!("and={and}, or={or}, not={not}");
}
```

## Characters

In Rust, `char` represents a 4-byte Unicode value — not just ASCII:

```rust
fn main() {
    let letter = 'a';
    let accent = 'ñ';
    let emoji  = '🦀'; // valid!

    println!("{letter} {accent} {emoji}");
}
```

> **Important:** strings (`str`, `String`) are different from `char`. A `char` is always a single Unicode character.

## Arithmetic operations

```rust
fn main() {
    let sum         = 5 + 10;
    let difference  = 95.5 - 4.3;
    let product     = 4 * 30;
    let quotient    = 56.7 / 32.2;
    let remainder   = 43 % 5;

    println!("{sum} {difference} {product} {quotient:.2} {remainder}");
}
```
