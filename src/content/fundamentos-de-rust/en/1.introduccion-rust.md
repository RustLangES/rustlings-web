# Introduction to Rust

Rust is a systems programming language designed to be **fast, safe, and concurrent**. It was initially created at Mozilla and its first stable release arrived in 2015.

## What is Rust used for?

Rust shines in contexts where performance and memory safety are critical:

- **Operating systems** — the Linux kernel already accepts contributions in Rust.
- **WebAssembly** — compile Rust code to run it directly in the browser.
- **CLI tools** — `ripgrep`, `fd`, `bat`, and `exa` are written in Rust.
- **Web backend** — high-performance servers with Actix-web or Axum.
- **Embedded systems** — low-level code without a runtime or garbage collector.

## The basic ecosystem

Rust comes with three main tools you'll use daily:

### rustc — The compiler

`rustc` transforms your `.rs` code into a binary executable.

```bash
rustc main.rs   # compiles main.rs
./main          # runs the result
```

In practice, you'll almost never call `rustc` directly — Cargo does it for you.

### cargo — The project manager

Cargo is the Swiss Army knife of Rust:

```bash
cargo new my_project   # creates a new project
cargo run              # compiles and runs
cargo build            # only compiles
cargo test             # runs the tests
cargo check            # verifies it compiles without generating the binary
```

### Project structure

When you run `cargo new hello_world`, you get:

```
hello_world/
├── Cargo.toml     ← manifest: name, version, dependencies
└── src/
    └── main.rs    ← your source code
```

`Cargo.toml` looks like this:

```toml
[package]
name = "hello_world"
version = "0.1.0"
edition = "2021"

[dependencies]
```

And `src/main.rs` comes with the classic Hello World:

```rust
fn main() {
    println!("Hello, world!");
}
```

## Why learn Rust

- **No garbage collector** — the compiler manages memory automatically through ownership, at no runtime cost.
- **C/C++ performance** — zero-cost abstractions.
- **Fearless concurrency** — the type system prevents data races at compile time.
- **Modern ecosystem** — Cargo, integrated documentation, first-class tooling.

Rust has a steep learning curve, especially around *ownership* and the *borrow checker*. But once you understand it, you write more correct and efficient code from the first attempt.
