# Exercise: Borrowing in Practice

Implement the following functions using references correctly. None of them should take ownership of the data.

## Functions to implement

```rust
// Returns the first word of the string (up to the first space)
fn primera_palabra(s: &str) -> &str

// Counts the vowels (a, e, i, o, u) in the string
fn contar_vocales(s: &str) -> usize

// Reverses the String in place (modifies the original)
fn invertir(s: &mut String)
```

## Examples

```
primera_palabra("hola mundo") → "hola"
primera_palabra("rust")       → "rust"

contar_vocales("hola")        → 2
contar_vocales("rust")        → 1

invertir(&mut "hola")         → modifies to "aloh"
```

## Hints

- For `primera_palabra`: use the `.find(' ')` method or `.bytes()` + `.enumerate()` to find the space.
- For `contar_vocales`: iterate over `.chars()` and filter with `matches`.
- For `invertir`: convert to bytes, reverse with `.reverse()` and convert back with `String::from_utf8`.

> Remember: `primera_palabra` receives `&str` and returns `&str` — the returned reference points to the parameter's data. This is valid because the input lives longer than the return.
