# Exercise: Slices

Implement functions that operate on slices (`&[i32]` and `&str`) without taking ownership of the data.

## Functions to implement

```rust
// Returns true if the elements are in ascending order (non-decreasing)
fn esta_ordenado(v: &[i32]) -> bool

// Sums the elements of the slice in the range [desde, hasta)
fn suma_rango(v: &[i32], desde: usize, hasta: usize) -> i32

// Returns true if `texto` contains `patron` as a substring
fn contiene_subcadena(texto: &str, patron: &str) -> bool

// Counts the number of space-separated words
fn contar_palabras(texto: &str) -> usize
```

## Examples

```
esta_ordenado(&[1, 2, 3])    → true
esta_ordenado(&[1, 3, 2])    → false
esta_ordenado(&[])            → true

suma_rango(&[1,2,3,4,5], 1, 4) → 9  (2+3+4)
suma_rango(&[1,2,3,4,5], 0, 0) → 0

contiene_subcadena("hola mundo", "mundo") → true
contiene_subcadena("rust", "xyz")         → false

contar_palabras("hola mundo") → 2
contar_palabras("")           → 0
```

## Hints

- `esta_ordenado`: use `.windows(2)` to iterate consecutive pairs, verify that `pair[0] <= pair[1]`.
- `suma_rango`: use `&v[desde..hasta]` to create a sub-slice and then `.iter().sum()`.
- `contiene_subcadena`: use the `.contains(patron)` method of `&str`.
- `contar_palabras`: use `.split_whitespace().count()`.
