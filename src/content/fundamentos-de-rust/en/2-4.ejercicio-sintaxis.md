# Exercise: Calculations with Variables and Constants

Implement conversion and calculation functions using only the types and arithmetic operators you've learned.

## Available constants

```rust
const TASA_IVA: f64 = 0.21;   // 21%
const GRAVEDAD: f64 = 9.81;   // m/s²
```

## Functions to implement

```rust
// Calculates the final price applying VAT (use the TASA_IVA constant)
fn precio_con_iva(base: f64) -> f64

// Converts degrees Celsius to Fahrenheit: F = C × 9/5 + 32
fn celsius_a_fahrenheit(c: f64) -> f64

// Kinetic energy: E = ½ × mass × velocity²
fn energia_cinetica(masa: f64, velocidad: f64) -> f64

// Converts total seconds into (minutes, remaining_seconds)
fn segundos_a_minutos_y_segundos(total_seg: u64) -> (u64, u64)
```

## Examples

```
precio_con_iva(100.0) → 121.0
precio_con_iva(50.0)  → 60.5

celsius_a_fahrenheit(0.0)   → 32.0
celsius_a_fahrenheit(100.0) → 212.0

energia_cinetica(2.0, 3.0)  → 9.0   (0.5 * 2 * 3² = 9)
energia_cinetica(10.0, 2.0) → 20.0

segundos_a_minutos_y_segundos(90)  → (1, 30)
segundos_a_minutos_y_segundos(125) → (2, 5)
```

## Hints

- `precio_con_iva`: multiply by `1.0 + TASA_IVA`.
- `celsius_a_fahrenheit`: `c * 9.0 / 5.0 + 32.0`.
- `energia_cinetica`: `0.5 * masa * velocidad * velocidad`.
- `segundos_a_minutos_y_segundos`: use `/` for minutes and `%` for remaining seconds.
