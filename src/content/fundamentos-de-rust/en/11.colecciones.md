# Basic Collections

Rust's standard library includes collections that store data on the heap and can grow dynamically. The most important ones are:

- **`Vec<T>`** — ordered list of elements of the same type
- **`String`** — dynamic text string (UTF-8)
- **`HashMap<K, V>`** — key-value hash table

Unlike arrays (`[T; N]`), these collections don't have a fixed size at compile time.

This chapter covers:

- `Vec<T>` — creation, access, modification
- `String` — construction and common operations
- `HashMap<K, V>` — insertion, search, update
- Basic iterators: `iter()`, `into_iter()`, `iter_mut()`
- Collections exercise
