# Vec<T> — Vectors

A `Vec<T>` is a dynamic list that grows automatically. It's the most used collection in Rust.

## Creating vectors

```rust
fn main() {
    // Empty Vec
    let mut v: Vec<i32> = Vec::new();
    v.push(1);
    v.push(2);
    v.push(3);

    // With the vec! macro
    let v2 = vec![1, 2, 3, 4, 5];

    // Vec with pre-allocated capacity (optimization)
    let mut v3: Vec<String> = Vec::with_capacity(10);
}
```

## Accessing elements

```rust
fn main() {
    let v = vec![10, 20, 30, 40, 50];

    // By index (panic if out of range)
    println!("{}", v[2]); // 30

    // get() — safe access with Option
    match v.get(2) {
        Some(x) => println!("Element: {x}"),
        None    => println!("Out of range"),
    }

    // First and last
    println!("{:?} {:?}", v.first(), v.last()); // Some(10) Some(50)
}
```

## Modifying vectors

```rust
fn main() {
    let mut v = vec![1, 2, 3];

    v.push(4);         // appends to the end
    v.insert(1, 10);   // inserts at index 1
    let last = v.pop(); // removes from the end → Option<i32>
    v.remove(0);       // removes at index 0

    println!("{:?}", v);
    println!("pop: {:?}", last);
}
```

## Iteration

```rust
fn main() {
    let v = vec![1, 2, 3, 4, 5];

    // Iteration by reference (doesn't consume v)
    for x in &v {
        print!("{x} ");
    }
    println!();

    // Mutable iteration
    let mut v2 = vec![1, 2, 3];
    for x in &mut v2 {
        *x *= 2;
    }
    println!("{:?}", v2); // [2, 4, 6]

    // Consuming the vector
    for x in v {    // v is moved here
        print!("{x} ");
    }
    // v no longer exists
}
```

## Common operations

```rust
fn main() {
    let numbers = vec![3, 1, 4, 1, 5, 9, 2, 6];

    println!("length:     {}", numbers.len());
    println!("empty:      {}", numbers.is_empty());
    println!("contains 5: {}", numbers.contains(&5));
    println!("sum:        {}", numbers.iter().sum::<i32>());
    println!("max:        {:?}", numbers.iter().max());

    // Sort (requires cloning if you want to keep the original)
    let mut sorted = numbers.clone();
    sorted.sort();
    println!("sorted: {:?}", sorted);

    // Filter and collect
    let evens: Vec<i32> = numbers.iter()
        .filter(|&&x| x % 2 == 0)
        .cloned()
        .collect();
    println!("evens: {:?}", evens);
}
```

## Vec with mixed types via enums

```rust
#[derive(Debug)]
enum Cell {
    Text(String),
    Number(f64),
    Boolean(bool),
}

fn main() {
    let row = vec![
        Cell::Text(String::from("Alice")),
        Cell::Number(30.0),
        Cell::Boolean(true),
    ];

    for cell in &row {
        println!("{:?}", cell);
    }
}
```
