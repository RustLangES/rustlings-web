# Control Flow

Rust's control structures share names with other languages (`if`, `while`, `for`, `loop`) but have an important twist: **almost everything in Rust is an expression**.

This means that `if`, `match`, and `loop` can return values, which allows for more concise and expressive code.

This chapter covers:

- `if` as an expression — not just a statement
- `match` — Rust's exhaustive pattern matching
- Loops: `loop`, `while`, and `for`
- `break` with a return value
