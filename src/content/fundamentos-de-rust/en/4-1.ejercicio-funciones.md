# Exercise: Mathematical Functions

Implement three classic functions using only what you know about functions, loops, and conditionals.

## Functions to implement

```rust
// Calculates n! (factorial)
// 0! = 1, n! = n × (n-1)!
fn factorial(n: u64) -> u64

// Returns true if n is prime
// (only divisible by 1 and itself)
fn es_primo(n: u64) -> bool

// Returns the n-th Fibonacci number
// fib(0) = 0, fib(1) = 1, fib(n) = fib(n-1) + fib(n-2)
fn fibonacci(n: u32) -> u64
```

## Examples

```
factorial(0)  → 1
factorial(5)  → 120
factorial(10) → 3_628_800

es_primo(2)  → true
es_primo(9)  → false
es_primo(97) → true

fibonacci(0)  → 0
fibonacci(1)  → 1
fibonacci(10) → 55
```

## Hints

- `factorial`: use a `for` loop with an accumulator. Remember that `0! = 1`.
- `es_primo`: for `n < 2` return `false`. Then check divisors from 2 to `√n` (or up to `n/2` if you don't want the square root). Use a loop with early return.
- `fibonacci`: use two variables `a` and `b` that you swap on each iteration — you don't need recursion.
