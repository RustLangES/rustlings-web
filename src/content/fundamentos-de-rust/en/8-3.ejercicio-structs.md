# Exercise: Struct with Methods

Implement a bounded `Contador` — a counter that cannot exceed a maximum or go below a minimum.

## Methods to implement

```rust
fn nuevo(minimo: i32, maximo: i32) -> Contador  // starts at minimum
fn incrementar(&mut self)                        // adds 1, without exceeding maximum
fn decrementar(&mut self)                        // subtracts 1, without going below minimum
fn valor(&self) -> i32                           // returns the current value
fn en_maximo(&self) -> bool                      // true if value == maximum
fn en_minimo(&self) -> bool                      // true if value == minimum
fn reiniciar(&mut self)                          // resets to minimum
```

## Examples

```
nuevo(0, 3)  → value = 0, en_minimo = true

incrementar x3 → value = 3, en_maximo = true
incrementar    → value stays at 3 (doesn't exceed maximum)

reiniciar → value = 0
```

## Hints

- The `valor` field starts at `minimo` when constructing.
- `incrementar`: use an `if` to not go past `self.maximo`.
- `decrementar`: use an `if` to not go below `self.minimo`.
- `reiniciar`: assign `self.minimo` to `self.valor`.
