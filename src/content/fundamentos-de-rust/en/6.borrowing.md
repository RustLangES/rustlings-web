# Borrowing: Safe References

*Borrowing* allows you to use a value without taking its ownership. It's like borrowing something — you can use it, but you must return it to its owner.

Without borrowing, you'd have to move or clone all the time, which would be inconvenient and inefficient.

This chapter covers:

- Immutable references `&T`
- Mutable references `&mut T`
- The borrow checker rules
- Common errors and how to resolve them
- The borrowing exercise
