# Exercise: Text Analysis

Implement three functions to analyze text using `HashMap` and `Vec`.

## Functions to implement

```rust
// Counts how many times each word appears in the text
fn frecuencia_palabras(texto: &str) -> HashMap<String, usize>

// Returns the n most frequent words, sorted from most to least frequent
fn top_n(frecuencias: &HashMap<String, usize>, n: usize) -> Vec<(&String, &usize)>

// Calculates the average of a list of numbers
// Returns None if the list is empty
fn promedio(numeros: &[f64]) -> Option<f64>
```

## Examples

```
frecuencia_palabras("hola mundo hola")
→ {"hola": 2, "mundo": 1}

top_n(&frequencies, 2) with {"a": 3, "b": 2, "c": 1}
→ [("a", 3), ("b", 2)]

promedio(&[1.0, 2.0, 3.0]) → Some(2.0)
promedio(&[])               → None
```

## Hints

- For `frecuencia_palabras`: use `split_whitespace()` to split words and `entry().or_insert(0)` to count them.
- For `top_n`: convert the `HashMap` to a `Vec` with `.iter().collect()`, sort with `.sort_by(...)` and take the first `n` with `.into_iter().take(n).collect()`.
- For `promedio`: `slice.iter().sum::<f64>()` and `slice.len()`.
