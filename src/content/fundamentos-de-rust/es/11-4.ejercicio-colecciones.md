# Ejercicio: Análisis de Texto

Implementa tres funciones para analizar texto usando `HashMap` y `Vec`.

## Funciones a implementar

```rust
// Cuenta cuántas veces aparece cada palabra en el texto
fn frecuencia_palabras(texto: &str) -> HashMap<String, usize>

// Retorna las n palabras más frecuentes, ordenadas de mayor a menor
fn top_n(frecuencias: &HashMap<String, usize>, n: usize) -> Vec<(&String, &usize)>

// Calcula el promedio de una lista de números
// Retorna None si la lista está vacía
fn promedio(numeros: &[f64]) -> Option<f64>
```

## Ejemplos

```
frecuencia_palabras("hola mundo hola")
→ {"hola": 2, "mundo": 1}

top_n(&frecuencias, 2) con {"a": 3, "b": 2, "c": 1}
→ [("a", 3), ("b", 2)]

promedio(&[1.0, 2.0, 3.0]) → Some(2.0)
promedio(&[])               → None
```

## Pistas

- Para `frecuencia_palabras`: usa `split_whitespace()` para separar palabras y `entry().or_insert(0)` para contarlas.
- Para `top_n`: convierte el `HashMap` en un `Vec` con `.iter().collect()`, ordena con `.sort_by(...)` y toma los primeros `n` con `.into_iter().take(n).collect()`.
- Para `promedio`: `slice.iter().sum::<f64>()` y `slice.len()`.
