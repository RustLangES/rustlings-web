# Colecciones Básicas

La librería estándar de Rust incluye colecciones que almacenan datos en el heap y pueden crecer dinámicamente. Las más importantes son:

- **`Vec<T>`** — lista ordenada de elementos del mismo tipo
- **`String`** — cadena de texto dinámica (UTF-8)
- **`HashMap<K, V>`** — tabla hash de clave-valor

A diferencia de los arrays (`[T; N]`), estas colecciones no tienen tamaño fijo en compilación.

Este capítulo cubre:

- `Vec<T>` — creación, acceso, modificación
- `String` — construcción y operaciones comunes
- `HashMap<K, V>` — inserción, búsqueda, actualización
- Iteradores básicos: `iter()`, `into_iter()`, `iter_mut()`
- Ejercicio de colecciones
