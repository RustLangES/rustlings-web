# Tipos Primitivos

Rust es un lenguaje de **tipado estático** — todos los tipos se conocen en tiempo de compilación. El compilador puede inferirlos en la mayoría de casos, pero es importante entenderlos.

## Enteros

Rust ofrece enteros con y sin signo de varios tamaños:

| Longitud | Con signo | Sin signo |
|----------|-----------|-----------|
| 8-bit    | `i8`      | `u8`      |
| 16-bit   | `i16`     | `u16`     |
| 32-bit   | `i32`     | `u32`     |
| 64-bit   | `i64`     | `u64`     |
| arch     | `isize`   | `usize`   |

```rust
fn main() {
    let a: i32 = -100;      // por defecto para enteros
    let b: u8  = 255;       // 0 a 255
    let c: u64 = 1_000_000; // guiones bajos para legibilidad
    let indice: usize = 0;  // siempre para indexar colecciones
}
```

**Regla práctica:** usa `i32` para enteros de propósito general y `usize` para índices.

## Punto flotante

```rust
fn main() {
    let x = 2.0;       // f64 por defecto
    let y: f32 = 3.0;  // f32 si necesitas ahorrar memoria

    let resultado = x * 3.14159;
    println!("{:.2}", resultado); // dos decimales
}
```

## Booleanos

```rust
fn main() {
    let t = true;
    let f: bool = false;

    let and = t && f;  // false
    let or  = t || f;  // true
    let not = !t;      // false

    println!("and={and}, or={or}, not={not}");
}
```

## Caracteres

En Rust, `char` representa un valor Unicode de 4 bytes — no solo ASCII:

```rust
fn main() {
    let letra = 'a';
    let acento = 'ñ';
    let emoji  = '🦀'; // ¡válido!

    println!("{letra} {acento} {emoji}");
}
```

> **Importante:** las cadenas de texto (`str`, `String`) son diferentes de `char`. Un `char` es siempre un único carácter Unicode.

## Operaciones aritméticas

```rust
fn main() {
    let suma        = 5 + 10;
    let diferencia  = 95.5 - 4.3;
    let producto    = 4 * 30;
    let cociente    = 56.7 / 32.2;
    let residuo     = 43 % 5;

    println!("{suma} {diferencia} {producto} {cociente:.2} {residuo}");
}
```
