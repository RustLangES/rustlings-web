# Bucles: loop, while y for

Rust tiene tres tipos de bucles, cada uno con un propósito diferente.

## `loop` — Bucle infinito

`loop` crea un bucle que solo termina con `break`. Es útil cuando no sabes de antemano cuántas veces iterar:

```rust
fn main() {
    let mut contador = 0;

    loop {
        contador += 1;
        if contador == 3 { continue; }  // salta esta iteración
        println!("contador: {contador}");
        if contador == 5 { break; }
    }
}
```

### `loop` como expresión — `break` con valor

```rust
fn main() {
    let mut intentos = 0;

    let resultado = loop {
        intentos += 1;
        if intentos == 10 {
            break intentos * 2;  // retorna el valor
        }
    };

    println!("Resultado: {resultado}"); // 20
}
```

## `while` — Bucle condicional

```rust
fn main() {
    let mut numero = 3;

    while numero != 0 {
        println!("{numero}!");
        numero -= 1;
    }
    println!("¡DESPEGUE!");
}
```

## `for` — Iteración sobre colecciones

`for` es el bucle más idiomático en Rust:

```rust
fn main() {
    let array = [10, 20, 30, 40, 50];

    for elemento in array {
        println!("{elemento}");
    }
}
```

### Rangos con `for`

```rust
fn main() {
    for i in 1..4 {    // 1, 2, 3 (excluye el 4)
        println!("{i}");
    }

    for i in 1..=4 {   // 1, 2, 3, 4 (incluye el 4)
        println!("{i}");
    }

    // Inverso
    for i in (1..=4).rev() {
        println!("{i}"); // 4, 3, 2, 1
    }
}
```

### Iterar con índices

```rust
fn main() {
    let frutas = ["manzana", "banana", "cereza"];

    for (indice, fruta) in frutas.iter().enumerate() {
        println!("{indice}: {fruta}");
    }
}
```

## Etiquetas para bucles anidados

Si tienes bucles anidados y necesitas hacer `break` del externo:

```rust
fn main() {
    'externo: for i in 0..5 {
        for j in 0..5 {
            if i == 2 && j == 2 {
                break 'externo; // sale del bucle externo
            }
            println!("({i}, {j})");
        }
    }
}
```

> **Regla práctica:**
> - Usa `for` cuando iteras sobre una colección o rango conocido.
> - Usa `while` cuando la condición de salida depende de un estado.
> - Usa `loop` cuando necesitas retornar un valor del bucle o la condición de salida es compleja.
