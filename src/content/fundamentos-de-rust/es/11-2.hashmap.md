# HashMap<K, V>

Un `HashMap` almacena pares clave-valor. Es la tabla hash de Rust.

## Crear y poblar

```rust
use std::collections::HashMap;

fn main() {
    let mut mapa: HashMap<String, i32> = HashMap::new();

    mapa.insert(String::from("uno"),  1);
    mapa.insert(String::from("dos"),  2);
    mapa.insert(String::from("tres"), 3);

    // Desde dos vectores con zip
    let claves = vec!["a", "b", "c"];
    let valores = vec![1, 2, 3];
    let mapa2: HashMap<_, _> = claves.iter().zip(valores.iter()).collect();
}
```

## Acceso a valores

```rust
fn main() {
    let mut m = HashMap::new();
    m.insert("clave", 42);

    // get — retorna Option<&V>
    match m.get("clave") {
        Some(v) => println!("Valor: {v}"),
        None    => println!("No encontrado"),
    }

    // Acceso directo (panic si no existe)
    println!("{}", m["clave"]); // 42
}
```

## Actualizaciones comunes

```rust
fn main() {
    let mut m: HashMap<String, u32> = HashMap::new();

    // Sobrescribir
    m.insert(String::from("x"), 10);
    m.insert(String::from("x"), 20); // sobrescribe

    // Insertar solo si no existe
    m.entry(String::from("y")).or_insert(5);
    m.entry(String::from("y")).or_insert(99); // y ya existe, no cambia

    // Actualizar basándose en el valor anterior
    let contador = m.entry(String::from("z")).or_insert(0);
    *contador += 1;

    println!("{:?}", m);
}
```

## Caso práctico: contar palabras

```rust
use std::collections::HashMap;

fn contar_palabras(texto: &str) -> HashMap<&str, usize> {
    let mut conteo = HashMap::new();

    for palabra in texto.split_whitespace() {
        let c = conteo.entry(palabra).or_insert(0);
        *c += 1;
    }

    conteo
}

fn main() {
    let texto = "hola mundo hola rust mundo hola";
    let conteo = contar_palabras(texto);

    let mut pares: Vec<_> = conteo.iter().collect();
    pares.sort_by(|a, b| b.1.cmp(a.1)); // ordenar por frecuencia

    for (palabra, n) in pares {
        println!("{palabra}: {n}");
    }
}
```

## Ownership en HashMaps

Los tipos que no implementan `Copy` se mueven al insertarse:

```rust
fn main() {
    let mut m = HashMap::new();

    let clave  = String::from("clave");
    let valor  = String::from("valor");

    m.insert(clave, valor);

    // clave y valor ya no existen — fueron movidos
    // println!("{clave}"); // ERROR

    // Solución: clonar si necesitas mantener las variables
    let k = String::from("k");
    m.insert(k.clone(), String::from("v"));
    println!("{k}"); // OK
}
```

## Eliminar entradas

```rust
fn main() {
    let mut m = HashMap::new();
    m.insert("a", 1);
    m.insert("b", 2);

    m.remove("a");
    println!("{:?}", m); // {"b": 2}

    m.retain(|_, v| *v > 1); // mantiene solo los que cumplen la condición
    println!("{:?}", m);
}
```
