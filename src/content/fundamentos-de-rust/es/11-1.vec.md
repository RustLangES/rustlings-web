# Vec<T> — Vectores

Un `Vec<T>` es una lista dinámica que crece automáticamente. Es la colección más usada en Rust.

## Crear vectores

```rust
fn main() {
    // Vec vacío
    let mut v: Vec<i32> = Vec::new();
    v.push(1);
    v.push(2);
    v.push(3);

    // Con la macro vec!
    let v2 = vec![1, 2, 3, 4, 5];

    // Vec con capacidad preasignada (optimización)
    let mut v3: Vec<String> = Vec::with_capacity(10);
}
```

## Acceso a elementos

```rust
fn main() {
    let v = vec![10, 20, 30, 40, 50];

    // Por índice (panic si fuera de rango)
    println!("{}", v[2]); // 30

    // get() — acceso seguro con Option
    match v.get(2) {
        Some(x) => println!("Elemento: {x}"),
        None    => println!("Fuera de rango"),
    }

    // Primer y último
    println!("{:?} {:?}", v.first(), v.last()); // Some(10) Some(50)
}
```

## Modificar vectores

```rust
fn main() {
    let mut v = vec![1, 2, 3];

    v.push(4);         // añade al final
    v.insert(1, 10);   // inserta en índice 1
    let ultimo = v.pop(); // extrae del final → Option<i32>
    v.remove(0);       // elimina en índice 0

    println!("{:?}", v);
    println!("pop: {:?}", ultimo);
}
```

## Iteración

```rust
fn main() {
    let v = vec![1, 2, 3, 4, 5];

    // Iteración por referencia (no consume v)
    for x in &v {
        print!("{x} ");
    }
    println!();

    // Iteración mutable
    let mut v2 = vec![1, 2, 3];
    for x in &mut v2 {
        *x *= 2;
    }
    println!("{:?}", v2); // [2, 4, 6]

    // Consumiendo el vector
    for x in v {    // v se mueve aquí
        print!("{x} ");
    }
    // v ya no existe
}
```

## Operaciones comunes

```rust
fn main() {
    let numeros = vec![3, 1, 4, 1, 5, 9, 2, 6];

    println!("longitud:  {}", numeros.len());
    println!("vacío:     {}", numeros.is_empty());
    println!("contiene 5: {}", numeros.contains(&5));
    println!("suma:      {}", numeros.iter().sum::<i32>());
    println!("máximo:    {:?}", numeros.iter().max());

    // Ordenar (requiere clonar si quieres mantener el original)
    let mut ordenado = numeros.clone();
    ordenado.sort();
    println!("ordenado: {:?}", ordenado);

    // Filtrar y recoger
    let pares: Vec<i32> = numeros.iter()
        .filter(|&&x| x % 2 == 0)
        .cloned()
        .collect();
    println!("pares: {:?}", pares);
}
```

## Vec con tipos mixtos mediante enums

```rust
#[derive(Debug)]
enum Celda {
    Texto(String),
    Numero(f64),
    Booleano(bool),
}

fn main() {
    let fila = vec![
        Celda::Texto(String::from("Alice")),
        Celda::Numero(30.0),
        Celda::Booleano(true),
    ];

    for celda in &fila {
        println!("{:?}", celda);
    }
}
```
