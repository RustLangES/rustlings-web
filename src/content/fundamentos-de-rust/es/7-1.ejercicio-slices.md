# Ejercicio: Slices

Implementa funciones que operen sobre slices (`&[i32]` y `&str`) sin tomar ownership de los datos.

## Funciones a implementar

```rust
// Retorna true si los elementos están en orden ascendente (no decreciente)
fn esta_ordenado(v: &[i32]) -> bool

// Suma los elementos del slice en el rango [desde, hasta)
fn suma_rango(v: &[i32], desde: usize, hasta: usize) -> i32

// Retorna true si `texto` contiene `patron` como subcadena
fn contiene_subcadena(texto: &str, patron: &str) -> bool

// Cuenta el número de palabras separadas por espacios
fn contar_palabras(texto: &str) -> usize
```

## Ejemplos

```
esta_ordenado(&[1, 2, 3])    → true
esta_ordenado(&[1, 3, 2])    → false
esta_ordenado(&[])            → true

suma_rango(&[1,2,3,4,5], 1, 4) → 9  (2+3+4)
suma_rango(&[1,2,3,4,5], 0, 0) → 0

contiene_subcadena("hola mundo", "mundo") → true
contiene_subcadena("rust", "xyz")         → false

contar_palabras("hola mundo") → 2
contar_palabras("")           → 0
```

## Pistas

- `esta_ordenado`: usa `.windows(2)` para iterar pares consecutivos, verifica que `par[0] <= par[1]`.
- `suma_rango`: usa `&v[desde..hasta]` para crear un sub-slice y luego `.iter().sum()`.
- `contiene_subcadena`: usa el método `.contains(patron)` de `&str`.
- `contar_palabras`: usa `.split_whitespace().count()`.
