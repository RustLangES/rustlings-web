# Ejercicio: Borrowing en Práctica

Implementa las siguientes funciones usando referencias correctamente. Ninguna de ellas debe tomar ownership de los datos.

## Funciones a implementar

```rust
// Retorna la primera palabra de la cadena (hasta el primer espacio)
fn primera_palabra(s: &str) -> &str

// Cuenta las vocales (a, e, i, o, u) en la cadena
fn contar_vocales(s: &str) -> usize

// Invierte el String en su lugar (modifica el original)
fn invertir(s: &mut String)
```

## Ejemplos

```
primera_palabra("hola mundo") → "hola"
primera_palabra("rust")       → "rust"

contar_vocales("hola")        → 2
contar_vocales("rust")        → 1

invertir(&mut "hola")         → modifica a "aloh"
```

## Pistas

- Para `primera_palabra`: usa el método `.find(' ')` o `.bytes()` + `.enumerate()` para encontrar el espacio.
- Para `contar_vocales`: itera sobre `.chars()` y filtra con `matches`.
- Para `invertir`: convierte a bytes, invierte con `.reverse()` y reconvierte con `String::from_utf8`.

> Recuerda: `primera_palabra` recibe `&str` y retorna `&str` — la referencia de retorno apunta a los datos del parámetro. Esto es válido porque la entrada vive más que el retorno.
