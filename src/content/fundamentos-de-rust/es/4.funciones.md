# Funciones

Las funciones son el bloque de construcción fundamental de cualquier programa Rust. Se definen con `fn` y por convención usan `snake_case`.

## Función básica

```rust
fn main() {
    println!("¡Hola, mundo!");
    otra_funcion();
}

fn otra_funcion() {
    println!("Esta es otra función.");
}
```

En Rust, el orden de declaración no importa — puedes llamar a `otra_funcion` antes de que aparezca en el código.

## Parámetros y tipos

Todos los parámetros **deben** tener tipo explícito:

```rust
fn saludar(nombre: &str, veces: u32) {
    for _ in 0..veces {
        println!("¡Hola, {nombre}!");
    }
}

fn main() {
    saludar("Alice", 3);
}
```

## Valor de retorno

El tipo de retorno se indica con `->`. La última expresión de la función es el valor de retorno (sin punto y coma):

```rust
fn cuadrado(x: i32) -> i32 {
    x * x  // expresión — sin punto y coma
}
```

Usar `return` explícito es válido para retornos anticipados:

```rust
fn dividir(a: f64, b: f64) -> f64 {
    if b == 0.0 {
        return 0.0;  // retorno anticipado
    }
    a / b  // retorno implícito
}
```

## Statements vs Expressions

Esta distinción es clave en Rust:

- **Statement (declaración):** realiza una acción, **no retorna valor**. Termina con `;`.
- **Expression (expresión):** evalúa a un valor. **Sin punto y coma**.

```rust
fn main() {
    let x = 5;              // statement

    let y = {
        let a = 3;
        a + 1               // expresión — retorna 4
    };                      // el bloque asigna 4 a y

    println!("y = {y}");    // y es 4
}
```

## Funciones que retornan `()`

Cuando una función no retorna nada, implícitamente retorna `()` (el "unit type"):

```rust
fn imprimir(mensaje: &str) {  // equivalente a -> ()
    println!("{mensaje}");
}
```

## Retornar múltiples valores con tuplas

Rust no tiene retornos múltiples directamente, pero las tuplas son idiomáticas para ello:

```rust
fn estadisticas(datos: &[f64]) -> (f64, f64, f64) {
    let suma: f64 = datos.iter().sum();
    let min = datos.iter().cloned().fold(f64::INFINITY, f64::min);
    let max = datos.iter().cloned().fold(f64::NEG_INFINITY, f64::max);
    (suma, min, max)
}

fn main() {
    let datos = [3.0, 1.5, 4.0, 1.0, 5.5];
    let (suma, min, max) = estadisticas(&datos);
    println!("suma={suma}, min={min}, max={max}");
}
```
