# Ejercicio: Calculadora con Manejo de Errores

Implementa tres operaciones matemáticas que retornan `Result` en lugar de hacer panic o producir resultados incorrectos silenciosamente.

## Funciones a implementar

```rust
// División entera — falla si b == 0
fn dividir(a: i32, b: i32) -> Result<i32, ErrorCalc>

// Raíz cuadrada — falla si n < 0
fn raiz(n: f64) -> Result<f64, ErrorCalc>

// Suma que detecta desbordamiento de i32
fn suma_segura(a: i32, b: i32) -> Result<i32, ErrorCalc>
```

## Pistas

- Para `dividir`: comprueba que `b != 0` antes de dividir.
- Para `raiz`: comprueba que `n >= 0.0` antes de llamar `.sqrt()`.
- Para `suma_segura`: usa el método `checked_add` de `i32`, que retorna `Option<i32>`.

```rust
// checked_add retorna None si hay desbordamiento
let resultado = a.checked_add(b); // Option<i32>
```

> Cuando las pruebas pasen, habrás implementado manejo de errores explícito y sin panics.
