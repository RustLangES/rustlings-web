# Control de Flujo

Las estructuras de control de Rust comparten nombres con otros lenguajes (`if`, `while`, `for`, `loop`) pero tienen un giro importante: **casi todo en Rust es una expresión**.

Esto significa que `if`, `match` y `loop` pueden retornar valores, lo que permite código más conciso y expresivo.

Este capítulo cubre:

- `if` como expresión — no solo como declaración
- `match` — el pattern matching exhaustivo de Rust
- Bucles: `loop`, `while` y `for`
- `break` con valor de retorno
