# if como Expresión

## if básico

Normalmente el `if` puede funcionar como una simple condición, si se cumple esta condición ejecuto este bloque de código.

```rust
fn main() {
    let numero = 6;

    if numero % 4 == 0 {
        println!("Divisible por 4");
    } else if numero % 3 == 0 {
        println!("Divisible por 3");
    } else if numero % 2 == 0 {
        println!("Divisible por 2");
    } else {
        println!("No divisible por 4, 3, ni 2");
    }
}
```

A diferencia de muchos lenguajes, la condición en Rust **no lleva paréntesis** y debe ser estrictamente `bool` — no hay conversión implícita de enteros:

```rust
// En C: if (x) { ... }  ← válido
// En Rust:
// if x { ... }          ← ERROR si x no es bool
// if x != 0 { ... }     ← correcto
```

## if como expresión

En Rust, `if` es una expresión — puede retornar un valor:

```rust
fn main() {
    let condicion = true;
    let numero = if condicion { 5 } else { 6 };
    println!("El número es: {numero}");
}
```

**Regla importante:** ambas ramas deben retornar el mismo tipo:

```rust
// ERROR: tipos incompatibles
// let valor = if condicion { 5 } else { "seis" };

// Correcto:
let valor = if condicion { 5 } else { 6 };
```

## Expresiones if en una línea vs multilínea

```rust
fn main() {
    let temperatura = 22;

    // Conciso para condiciones simples
    let estado = if temperatura > 20 { "cálido" } else { "frío" };

    // Multilínea para lógica más compleja
    let actividad = if temperatura > 30 {
        "ir a la playa"
    } else if temperatura > 20 {
        "dar un paseo"
    } else {
        "quedarse en casa"
    };

    println!("Clima {estado}: {actividad}");
}
```

> **Consejo:** Usa `if` como expresión para asignar valores basados en condiciones. Es mas idiomático que usar `let mut` con estas reasignaciones.
