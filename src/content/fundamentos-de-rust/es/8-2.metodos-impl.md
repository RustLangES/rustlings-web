# Métodos con `impl`

Los métodos se definen en bloques `impl` (implementation). Son funciones que tienen acceso a los datos del struct a través de `self`.

## Métodos de instancia

```rust
#[derive(Debug)]
struct Circulo {
    radio: f64,
}

impl Circulo {
    fn area(&self) -> f64 {
        std::f64::consts::PI * self.radio * self.radio
    }

    fn perimetro(&self) -> f64 {
        2.0 * std::f64::consts::PI * self.radio
    }
}

fn main() {
    let c = Circulo { radio: 5.0 };
    println!("área: {:.2}", c.area());
}
```

## `self`, `&self`, `&mut self`

El primer parámetro determina cómo el método accede al struct:

| Parámetro | Significado | Consume el struct |
|-----------|-------------|-------------------|
| `self` | Toma ownership | Sí |
| `&self` | Lee el struct | No |
| `&mut self` | Modifica el struct | No |

```rust
impl Rectangulo {
    fn area(&self) -> f64 {           // solo lee
        self.ancho * self.alto
    }

    fn escalar(&mut self, factor: f64) { // modifica
        self.ancho *= factor;
        self.alto  *= factor;
    }

    fn convertir_a_cadena(self) -> String { // consume
        format!("{}x{}", self.ancho, self.alto)
    }
}
```

## Funciones asociadas (constructores)

Las funciones que no toman `self` se llaman *associated functions*. Se usan para constructores y utilidades:

```rust
impl Rectangulo {
    // Constructor: se llama con Rectangulo::nuevo(...)
    fn nuevo(ancho: f64, alto: f64) -> Rectangulo {
        Rectangulo { ancho, alto }
    }

    fn cuadrado(lado: f64) -> Rectangulo {
        Rectangulo { ancho: lado, alto: lado }
    }
}

fn main() {
    let r = Rectangulo::nuevo(10.0, 5.0);
    let q = Rectangulo::cuadrado(4.0);
}
```

## Múltiples bloques `impl`

Puedes tener varios bloques `impl` para el mismo struct — son equivalentes a uno solo:

```rust
impl Rectangulo {
    fn area(&self) -> f64 { self.ancho * self.alto }
}

impl Rectangulo {
    fn puede_contener(&self, otro: &Rectangulo) -> bool {
        self.ancho >= otro.ancho && self.alto >= otro.alto
    }
}
```

## Encadenamiento de métodos (method chaining)

Si los métodos retornan `&mut Self`, puedes encadenarlos:

```rust
struct Constructor {
    nombre: String,
    valor:  i32,
}

impl Constructor {
    fn nuevo() -> Constructor {
        Constructor { nombre: String::new(), valor: 0 }
    }

    fn nombre(mut self, n: &str) -> Self {
        self.nombre = n.to_string();
        self
    }

    fn valor(mut self, v: i32) -> Self {
        self.valor = v;
        self
    }
}

fn main() {
    let c = Constructor::nuevo()
        .nombre("test")
        .valor(42);
    println!("{}: {}", c.nombre, c.valor);
}
```
