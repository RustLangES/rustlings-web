# Ejercicio: Funciones Matemáticas

Implementa tres funciones clásicas usando solo lo que sabes de funciones, bucles y condicionales.

## Funciones a implementar

```rust
// Calcula n! (factorial)
// 0! = 1, n! = n × (n-1)!
fn factorial(n: u64) -> u64

// Retorna true si n es primo
// (solo divisible por 1 y por sí mismo)
fn es_primo(n: u64) -> bool

// Retorna el n-ésimo número de Fibonacci
// fib(0) = 0, fib(1) = 1, fib(n) = fib(n-1) + fib(n-2)
fn fibonacci(n: u32) -> u64
```

## Ejemplos

```
factorial(0)  → 1
factorial(5)  → 120
factorial(10) → 3_628_800

es_primo(2)  → true
es_primo(9)  → false
es_primo(97) → true

fibonacci(0)  → 0
fibonacci(1)  → 1
fibonacci(10) → 55
```

## Pistas

- `factorial`: usa un bucle `for` con un acumulador. Recuerda que `0! = 1`.
- `es_primo`: para `n < 2` retorna `false`. Luego comprueba divisores desde 2 hasta `√n` (o hasta `n/2` si no quieres la raíz). Usa un bucle con retorno anticipado.
- `fibonacci`: usa dos variables `a` y `b` que vas intercambiando en cada iteración — no necesitas recursión.
