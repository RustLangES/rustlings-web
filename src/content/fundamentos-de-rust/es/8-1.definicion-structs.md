# Definición de Structs

## Struct básica

```rust
struct Usuario {
    nombre: String,
    email:  String,
    edad:   u32,
    activo: bool,
}
```

Para crear una instancia, especifica todos los campos:

```rust
fn main() {
    let usuario = Usuario {
        nombre: String::from("Alice"),
        email:  String::from("alice@ejemplo.com"),
        edad:   30,
        activo: true,
    };

    println!("Nombre: {}", usuario.nombre);
    println!("Edad:   {}", usuario.edad);
}
```

## Acceso y mutabilidad

Para modificar campos, la instancia completa debe ser `mut`:

```rust
fn main() {
    let mut u = Usuario {
        nombre: String::from("Bob"),
        email:  String::from("bob@ejemplo.com"),
        edad:   25,
        activo: true,
    };

    u.edad = 26;    // OK: u es mut
    u.activo = false;
    println!("{}: {}", u.nombre, u.edad);
}
```

No puedes marcar solo un campo como mutable — es todo el struct o nada.

## Field init shorthand

Cuando el nombre del parámetro coincide con el campo, puedes omitir la repetición:

```rust
fn crear_usuario(nombre: String, email: String) -> Usuario {
    Usuario {
        nombre,   // en lugar de nombre: nombre
        email,    // en lugar de email: email
        edad: 18,
        activo: true,
    }
}
```

## Struct update syntax

Crea una instancia nueva basándote en otra, cambiando solo algunos campos:

```rust
fn main() {
    let u1 = Usuario {
        nombre: String::from("Alice"),
        email:  String::from("alice@ejemplo.com"),
        edad:   30,
        activo: true,
    };

    let u2 = Usuario {
        email: String::from("alice_nuevo@ejemplo.com"),
        ..u1   // toma el resto de u1
    };

    // Nota: u1.nombre fue movido a u2, así que u1 ya no es completamente válido
    println!("{}", u2.nombre);
}
```

## Tipos de structs

### Tuple structs

```rust
struct Color(u8, u8, u8);
struct Punto3D(f64, f64, f64);

fn main() {
    let rojo   = Color(255, 0, 0);
    let origen = Punto3D(0.0, 0.0, 0.0);

    println!("R={} G={} B={}", rojo.0, rojo.1, rojo.2);

    let Color(r, g, b) = rojo; // destructuring
    println!("r={r}");
}
```

Son útiles para crear tipos con semántica diferente aunque tengan la misma estructura interna. `Color` y `Punto3D` son tipos distintos aunque ambos contengan tres valores.

### Unit structs

```rust
struct Marcador;

fn main() {
    let _m = Marcador;
}
```

Útiles para implementar traits sin necesitar datos.

## Debug con `#[derive(Debug)]`

Para poder imprimir un struct con `{:?}`, añade el derive:

```rust
#[derive(Debug)]
struct Rectangulo {
    ancho: u32,
    alto:  u32,
}

fn main() {
    let rect = Rectangulo { ancho: 30, alto: 50 };
    println!("{:?}", rect);   // Rectangulo { ancho: 30, alto: 50 }
    println!("{:#?}", rect);  // pretty-print
}
```
