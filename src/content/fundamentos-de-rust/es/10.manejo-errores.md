# Manejo Básico de Errores

Rust no tiene excepciones. En su lugar, usa `Result<T, E>` para errores recuperables y `panic!` para errores irrecuperables. Esta distinción es explícita y deliberada.

Este capítulo cubre:

- `Result<T, E>` en profundidad
- `match` con `Result`
- `unwrap` y `expect` — cuándo usarlos y cuándo no
- El operador `?` para propagar errores
- Errores personalizados
