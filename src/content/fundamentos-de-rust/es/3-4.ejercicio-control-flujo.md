# Ejercicio: Control de Flujo

Aplica `if`, `match` y bucles para resolver estos problemas usando solo tipos primitivos.

## Funciones a implementar

```rust
// Retorna -1 si n es negativo, 0 si es cero, 1 si es positivo
fn clasificar(n: i32) -> i32

// Suma todos los números pares de 1 hasta `limite` (inclusive)
fn suma_pares_hasta(limite: u32) -> u32

// Cuenta cuántos números de 1 hasta `limite` (inclusive) son divisibles por `factor`
fn contar_multiplos(limite: u32, factor: u32) -> u32

// Calcula base^exp usando un bucle (sin usar pow)
fn potencia(base: i64, exp: u32) -> i64
```

## Ejemplos

```
clasificar(-5) → -1
clasificar(0)  → 0
clasificar(3)  → 1

suma_pares_hasta(10) → 30   (2+4+6+8+10)
suma_pares_hasta(1)  → 0

contar_multiplos(10, 2) → 5  (2,4,6,8,10)
contar_multiplos(10, 3) → 3  (3,6,9)

potencia(2, 10) → 1024
potencia(5, 0)  → 1
```

## Pistas

- `clasificar`: usa `match n` con guardas (`n if n < 0 => -1`) o un `if/else if/else`.
- `suma_pares_hasta`: usa `for n in 1..=limite` y filtra con `if n % 2 == 0`.
- `contar_multiplos`: similar, incrementa un contador cuando `n % factor == 0`.
- `potencia`: empieza con `resultado = 1` y multiplica en un bucle `exp` veces.
