# Ejercicio: Enums en Acción

Dos ejercicios que usan enums con `match` exhaustivo y variantes con datos.

## Parte 1: Monedas

```rust
enum Moneda { Centavo, Cinco, Diez, Veinticinco }

// Retorna el valor en centavos de cada moneda
fn valor_centavos(moneda: &Moneda) -> u32

// Suma el valor total de un slice de monedas
fn total_centavos(monedas: &[Moneda]) -> u32
```

```
valor_centavos(&Moneda::Veinticinco) → 25
total_centavos(&[Veinticinco, Diez, Cinco, Centavo, Centavo]) → 42
```

## Parte 2: Temperatura

El enum `Temperatura` puede representar grados en tres escalas distintas:

```rust
enum Temperatura {
    Celsius(f64),
    Fahrenheit(f64),
    Kelvin(f64),
}

// Convierte cualquier temperatura a Celsius
fn a_celsius(temp: &Temperatura) -> f64

// Retorna true si la temperatura equivale a 0°C o menos
fn es_congelamiento(temp: &Temperatura) -> bool
```

Fórmulas de conversión:
```
Fahrenheit → Celsius:  (F - 32) × 5/9
Kelvin → Celsius:       K - 273.15
```

## Pistas

- `valor_centavos`: un `match` simple sobre las variantes.
- `total_centavos`: itera el slice con `for` y acumula con `valor_centavos`.
- `a_celsius`: usa `match temp` con patrones `Temperatura::Celsius(c) => *c`, etc.
- `es_congelamiento`: llama a `a_celsius(temp)` y compara con `0.0`.
