# Introducción a Rust

Rust es un lenguaje de programación de sistemas diseñado para ser **rápido, seguro y concurrente**. Fue creado inicialmente en Mozilla y su primera versión estable llegó en 2015.

## ¿Para qué sirve Rust?

Rust brilla en contextos donde el rendimiento y la seguridad de memoria son críticos:

- **Sistemas operativos** — el kernel de Linux ya acepta contribuciones en Rust.
- **WebAssembly** — compila código Rust para ejecutarlo directamente en el navegador.
- **Herramientas CLI** — `ripgrep`, `fd`, `bat` y `exa` están escritas en Rust.
- **Backend web** — servidores de alta performance con Actix-web o Axum.
- **Sistemas embebidos** — código de bajo nivel sin runtime ni recolector de basura.

## El ecosistema básico

Rust viene con tres herramientas principales que usarás a diario:

### rustc — El compilador

`rustc` transforma tu código `.rs` en un ejecutable binario.

```bash
rustc main.rs   # compila main.rs
./main          # ejecuta el resultado
```

En la práctica, casi nunca llamarás a `rustc` directamente — Cargo lo hace por ti.

### cargo — El gestor de proyectos

Cargo es la navaja suiza de Rust:

```bash
cargo new mi_proyecto   # crea un nuevo proyecto
cargo run               # compila y ejecuta
cargo build             # solo compila
cargo test              # ejecuta los tests
cargo check             # verifica que compila sin generar el binario
```

### La estructura de un proyecto

Cuando haces `cargo new hola_mundo`, obtienes:

```
hola_mundo/
├── Cargo.toml     ← manifiesto: nombre, versión, dependencias
└── src/
    └── main.rs    ← tu código fuente
```

`Cargo.toml` luce así:

```toml
[package]
name = "hola_mundo"
version = "0.1.0"
edition = "2021"

[dependencies]
```

Y `src/main.rs` viene con el clásico Hello World:

```rust
fn main() {
    println!("Hello, world!");
}
```

## Por qué aprender Rust

- **Sin garbage collector** — el compilador gestiona la memoria automáticamente mediante ownership, sin coste en tiempo de ejecución.
- **Rendimiento de C/C++** — zero-cost abstractions.
- **Concurrencia sin miedo** — el sistema de tipos previene data races en tiempo de compilación.
- **Ecosistema moderno** — Cargo, documentación integrada, tooling de primer nivel.

Rust tiene una curva de aprendizaje pronunciada, especialmente alrededor del *ownership* y el *borrow checker*. Pero una vez que lo entiendes, escribes código más correcto y eficiente desde el primer intento.

¡Empecemos!
