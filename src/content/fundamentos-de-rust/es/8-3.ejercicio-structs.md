# Ejercicio: Struct con Métodos

Implementa un `Contador` acotado — un contador que no puede superar un máximo ni bajar de un mínimo.

## Métodos a implementar

```rust
fn nuevo(minimo: i32, maximo: i32) -> Contador  // empieza en el mínimo
fn incrementar(&mut self)                        // suma 1, sin superar el máximo
fn decrementar(&mut self)                        // resta 1, sin bajar del mínimo
fn valor(&self) -> i32                           // retorna el valor actual
fn en_maximo(&self) -> bool                      // true si valor == maximo
fn en_minimo(&self) -> bool                      // true si valor == minimo
fn reiniciar(&mut self)                          // vuelve al mínimo
```

## Ejemplos

```
nuevo(0, 3)  → valor = 0, en_minimo = true

incrementar x3 → valor = 3, en_maximo = true
incrementar    → valor sigue siendo 3 (no supera el máximo)

reiniciar → valor = 0
```

## Pistas

- El campo `valor` empieza en `minimo` al construir.
- `incrementar`: usa un `if` para no pasar de `self.maximo`.
- `decrementar`: usa un `if` para no bajar de `self.minimo`.
- `reiniciar`: asigna `self.minimo` a `self.valor`.
