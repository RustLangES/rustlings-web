# Structs

Las structs son tipos de datos personalizados que agrupan campos relacionados bajo un nombre. Son el equivalente de las clases en otros lenguajes, aunque sin herencia.

En Rust, la lógica asociada a una struct se define por separado en bloques `impl`, lo que mantiene la definición de datos y el comportamiento claramente separados.

Este capítulo cubre:

- Definición de structs y creación de instancias
- Tuple structs y unit structs
- Bloques `impl`: métodos e funciones asociadas
- `self`, `&self`, `&mut self`
- Struct update syntax
- Ejercicio de structs
