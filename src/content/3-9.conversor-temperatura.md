# Ejercicio: Conversor de Temperatura

Implementa dos funciones para convertir entre Celsius y Fahrenheit.

## Fórmulas

```
°F = °C × 9/5 + 32
°C = (°F − 32) × 5/9
```

## Funciones a implementar

```rust
fn celsius_a_fahrenheit(celsius: f64) -> f64
fn fahrenheit_a_celsius(fahrenheit: f64) -> f64
```

## Ejemplos

```
celsius_a_fahrenheit(0.0)   → 32.0
celsius_a_fahrenheit(100.0) → 212.0
fahrenheit_a_celsius(32.0)  → 0.0
fahrenheit_a_celsius(212.0) → 100.0
```

> Implementa las funciones y ejecuta los tests para verificar tu solución.
