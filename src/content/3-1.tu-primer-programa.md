# Tu Primer Programa

```rust
fn main() {
    println!("¡Hola, mundo!");
}
```

Este simple programa ilustra varios conceptos importantes:
- `fn` define una función
- `main` es el punto de entrada del programa
- `println!` es un macro (nota el `!`)
- Los bloques de código van entre llaves `{}`
